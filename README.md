# 📦 Portfolio Website (Technical Details)

## Overview

This is my personal portfolio website built with modern web technologies to showcase my skills, projects, and experience.

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Milipatel3110/mili-portfolio.git

# Navigate to the project directory
cd mili-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

```
mili-portfolio/
├── src/
│   ├── app/              # Next.js App Router pages
│   ├── components/       # React components
│   │   ├── sections/    # Portfolio sections (Hero, About, Projects, Skills, etc.)
│   │   ├── common/      # Shared components (Navbar, Footer, etc.)
│   │   └── providers/   # Context providers (ThemeProvider)
│   ├── content/         # Data files (projects, skills, experience, etc.)
│   ├── hooks/          # Custom React hooks
│   └── lib/            # Utility functions
├── public/             # Static assets
│   ├── certificates/   # Certification images & PDFs
│   └── me.jpg          # Profile picture
└── ...
```

## 🎨 Customization

### Updating Content

- **Projects:** Edit `src/content/projects.ts`
- **Skills:** Edit `src/components/sections/Skills.tsx`
- **Experience:** Edit `src/content/experience.ts`
- **Education:** Edit `src/content/education.ts`
- **Profile:** Edit `src/content/profile.ts`

### Theme

The website supports both light and dark modes. Toggle is available in the navbar.

## 🚢 Deployment

Deploy to Vercel with zero configuration:

1. Push your code to GitHub
2. Import the project in Vercel
3. Deploy!

The site is live at: [https://www.miliipatel.com/](https://www.miliipatel.com/)
